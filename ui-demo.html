Rokkon Engine Dashboard - Button State Demo

Before Whitelisting:

This button appears for healthy module services that haven't been whitelisted yet.

After Whitelisting:

This button appears for modules that have been successfully whitelisted. It's disabled to show the status.

Implementation Details:

  • The dashboard now fetches both services and global modules
  • It checks if each module service is in the global registry
  • Button text and style changes based on whitelist status
  • Whitelisted modules show a green success button with checkmark
  • Non-whitelisted modules show an outline button with plus icon
