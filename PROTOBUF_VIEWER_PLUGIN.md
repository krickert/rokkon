# Protobuf Viewer Plugin for IntelliJ IDEA

## Overview

This document provides instructions for installing and using a plugin in JetBrains IntelliJ IDEA that allows for pretty visualization of Protocol Buffer (protobuf) binary files.

## Recommended Plugin: Protocol Buffer Editor

For viewing binary protobuf files (`.bin`) in IntelliJ IDEA, we recommend the **Protocol Buffer Editor** plugin by Jeremy Volkman.

### Features

- Syntax highlighting for `.proto` files
- Code completion and navigation for `.proto` files
- **Binary protobuf file viewer** - allows you to view the contents of binary protobuf files in a structured, human-readable format
- Support for editing and saving changes to binary protobuf files

### Installation Instructions

1. Open IntelliJ IDEA
2. Go to `File > Settings` (or `IntelliJ IDEA > Preferences` on macOS)
3. Select `Plugins` from the left sidebar
4. Click on the `Marketplace` tab
5. Search for "Protocol Buffer Editor"
6. Click the `Install` button next to the plugin
7. Restart IntelliJ IDEA when prompted

### Usage Instructions

#### Viewing Binary Protobuf Files

1. Navigate to a binary protobuf file (`.bin`) in your project
2. Double-click to open the file
3. The plugin will automatically detect the file as a binary protobuf and display it in a structured format
4. If the plugin cannot automatically detect the message type, you may need to specify it:
   - Right-click on the file
   - Select "Open as Protobuf"
   - Choose the appropriate message type from your project's protobuf definitions

#### Troubleshooting

If the plugin cannot automatically detect the message type:

1. Make sure your `.proto` files are properly compiled and the generated Java classes are in your project's classpath
2. Try specifying the message type manually when opening the file
3. Ensure the binary file was actually created using the protobuf format (check the `saveProtobufToDisk` method in `ProtobufUtils.java`)

## Alternative Plugins

### Protobuf Support by JetBrains

This official JetBrains plugin provides excellent support for `.proto` files but has limited functionality for viewing binary protobuf files.

### Protobuf Formatter

This plugin focuses on formatting `.proto` files and doesn't provide binary file viewing capabilities.

## Project-Specific Information

In this project, protobuf binary files are created using the `saveProtobufToDisk` and `saveProtobufsToDisk` methods in the `ProtobufUtils` class. These files have a `.bin` extension and contain serialized protobuf messages as defined in the project's `.proto` files located in `rokkon-protobuf/src/main/resources/`.

The main protobuf definitions are in:
- `rokkon_core_types.proto`
- `pipe_step_processor_service.proto`
- `module_registration.proto`
- And other related `.proto` files

## Conclusion

The Protocol Buffer Editor plugin provides the best solution for viewing binary protobuf files in IntelliJ IDEA. After installation, you should be able to open and inspect any protobuf binary files generated by the application, making debugging and development much easier.
